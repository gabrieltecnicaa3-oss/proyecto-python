CAMBIOS DETALLADOS
═════════════════════════════════════════════════════════════════════════════

1️⃣  CANTIDAD PERSONALIZADA (x de total)
─────────────────────────────────────

ANTES:
  Tabla mostraba solo checkboxes y descripción de piezas
  
DESPUÉS:
  Nueva columna "A Enviar" con input numérico
  Permite ingresar cantidad específica a enviar
  Valida que no exceda cantidad total
  
EJEMPLO:
  Total disponible: 18
  A Enviar (editable): [5] ← usuario ingresa 5
  En PDF se muestra: 5 de 18

CÓDIGO EN TABLA:
  <input type="number" name="cant_${pieza.id}" 
         class="cantidad-input" value="${cantidadTotal}" 
         min="0" max="${cantidadTotal}" placeholder="0">

─────────────────────────────────────

2️⃣  SIN DECIMALES
─────────────────

ANTES:
  Cantidades: 18.0, 1.0, 2.5, etc (con decimales)
  
DESPUÉS:
  Cantidades: 18, 1, 2 (enteros)
  Conversión: parseInt(parseFloat(cantidad))
  
EJEMPLO:
  BD: 18.0 → Web UI: 18 → PDF: 18 ✓
  BD: 1.0 → Web UI: 1 → PDF: 1 ✓

CÓDIGO:
  JavaScript: parseInt(parseFloat(pieza.cantidad) || 0)
  Python: int(float(cantidad) if cantidad else 0)

─────────────────────────────────────

3️⃣  CAMPO DE TRANSPORTE
────────────────────────

UBICACIÓN:
  Debajo del campo "Fecha de Remito"
  
TIPO:
  Input de texto (opcional)
  
PLACEHOLDER:
  "Ej: Empresa XYZ, Auto particular, etc."
  
USO EN PDF:
  Se muestra después del encabezado
  "Transporte: Empresa XYZ"
  
CÓDIGO HTML:
  <div class="form-group">
    <label>Transporte:</label>
    <input type="text" name="transporte" 
           placeholder="Ej: Empresa XYZ, Auto particular, etc.">
  </div>

  • Observaciones: TextArea para notas
